# Agents

This repository is for AI testing with a minimal Spring Boot web app.

## Goal
- Ensure the web service can start reliably.
- Provide simple endpoints for quick validation.

## Run
- `./mvnw spring-boot:run` (if wrapper exists)
- `mvn spring-boot:run`

## Verify
- Open `http://localhost:8080/`
- Open `http://localhost:8080/health`
