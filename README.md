# ai-springboot-test-1

Minimal Spring Boot Web application for AI testing.

## Requirements

- Java 17+
- Maven 3.9+

## Run

```bash
mvn spring-boot:run
```

## Endpoints

- `GET /` -> `ai-springboot-test-1 is running`
- `GET /health` -> `ok`

## Notes

- Default port: `8080`
- This repository includes `agents.md` for AI workflow testing.
